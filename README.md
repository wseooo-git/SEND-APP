# SEND

클라이밍 파티할 때 점수 관리하려고 만든 앱이에요.

크루원들이랑 팀 나눠서 볼더링 대결할 때 쓰면 좋아요. 링크 하나 공유하면 각자 폰으로 점수 입력하고, 실시간으로 팀 점수 확인할 수 있어요.

## 이런 기능이 있어요

- 6자리 코드로 파티 공유
- 담장, 더클라임, 클라이밍파크 등 주요 클라이밍장 난이도 프리셋
- 난이도 커스텀도 가능
- 드래그앤드롭으로 팀 배정
- 특정 사람들 같은 팀으로 묶기
- 본인 난이도 기준으로 도전/적정/기본 점수 자동 계산
- 상위 10개 문제만 집계 (설정 가능)
- 팀별 점수, 개인 순위 실시간 확인

---

## 우리 크루도 쓰고 싶어요

이 앱은 각 크루가 직접 배포해서 쓰는 방식이에요. 중앙 서버 없이 각자 운영하는 거라 데이터도 완전히 분리되고, 비용도 각자 부담 (근데 무료 범위 내에서 충분해요).

배포하면 그 크루 전용 앱이 생기고, 파티 만들기 누르면 그 사람이 관리자가 됩니다.

필요한 거:
- GitHub 계정
- Firebase 계정 (Google 계정 있으면 됨)
- Vercel 계정 (GitHub으로 가입)

전부 무료예요. 아래 배포 가이드 따라하면 30분이면 끝나요.

## 사용법

### 파티 만들기 (관리자)

1. 메인 화면에서 **파티 만들기** 선택
2. 파티 이름, 장소, 날짜 입력
3. 클라이밍장 프리셋 선택하거나 직접 난이도 설정
4. 팀 수, 최대 문제 수, 배율 설정
5. 파티 생성되면 **파티 코드** 나옴 → 크루원들한테 공유

### 참가하기

1. 메인 화면에서 **파티 참여하기** 선택
2. 공유받은 6자리 코드 입력
3. 본인 이름이랑 난이도 선택해서 참가 신청
4. 관리자가 승인하면 참가 완료

### 팀 배정 (관리자)

1. 로비에서 **팀 배정하기** 선택
2. 참가자 카드를 드래그해서 원하는 팀에 놓기
3. 특정 사람들 같은 팀 하고 싶으면 두 명 클릭해서 묶기
4. **자동 배정** 누르면 알아서 균형 맞춰줌
5. 다 됐으면 **배정 완료**

### 점수 입력

1. 게임 화면에서 본인 이름 선택
2. 문제 완등하면 해당 난이도 버튼 터치
3. 도전(+1단계)은 1.5배, 적정은 1배, 기본(-1단계)은 0.5배
4. 잘못 입력했으면 기록에서 삭제 가능
5. 10개 넘게 입력해도 점수 높은 10개만 집계됨

### 현황 보기

- 오른쪽 위 트로피 아이콘 누르면 현황판
- 팀별 총점, 개인 순위 실시간으로 확인

---

## 점수 계산 방식

본인 난이도 기준으로:

| 구분 | 설명 | 배율 |
|------|------|------|
| 도전 | 한 단계 위 난이도 | ×1.5 |
| 적정 | 본인 난이도 | ×1.0 |
| 기본 | 한 단계 아래 난이도 | ×0.5 |

예를 들어 보라가 적정인 사람이 흰색(한 단계 위) 문제 풀면 도전으로 1.5배 점수.

## 설치 / 배포

### 1단계: 이 저장소 Fork

1. 이 페이지 오른쪽 위에 **Fork** 버튼 클릭
2. 본인 GitHub 계정으로 복사됨

### 2단계: Firebase 설정

1. [Firebase Console](https://console.firebase.google.com) 접속 (Google 로그인)
2. **프로젝트 추가** 클릭
3. 프로젝트 이름 입력 (예: `our-climbing-crew`)
4. Google Analytics는 사용 안함으로 해도 됨 → **프로젝트 만들기**

#### Realtime Database 만들기
1. 왼쪽 메뉴 **빌드** → **Realtime Database**
2. **데이터베이스 만들기** 클릭
3. 위치: **싱가포르** 선택 (한국에서 제일 빠름)
4. **테스트 모드에서 시작** → **사용 설정**

#### 웹 앱 등록
1. 프로젝트 개요에서 **</>** (웹 아이콘) 클릭
2. 앱 닉네임 아무거나 입력
3. **앱 등록** 클릭
4. `firebaseConfig` 값들이 나옴 → 메모장에 복사해두기

### 3단계: Vercel 배포

1. [Vercel](https://vercel.com) 접속 → **Continue with GitHub**
2. Fork한 저장소 **Import**
3. **Environment Variables** 섹션에 Firebase 값 입력:

```
VITE_FIREBASE_API_KEY = (apiKey 값)
VITE_FIREBASE_AUTH_DOMAIN = (authDomain 값)
VITE_FIREBASE_DATABASE_URL = (databaseURL 값)
VITE_FIREBASE_PROJECT_ID = (projectId 값)
VITE_FIREBASE_STORAGE_BUCKET = (storageBucket 값)
VITE_FIREBASE_MESSAGING_SENDER_ID = (messagingSenderId 값)
VITE_FIREBASE_APP_ID = (appId 값)
```

4. **Deploy** 클릭
5. 1-2분 후 배포 완료 → 링크 생성!

이 링크를 크루원들한테 공유하면 됩니다.

클라이밍 크루에서 볼더링 파티할 때마다 점수 집계가 귀찮았어요. 엑셀로 하자니 실시간 공유가 안 되고, 구글 시트는 입력하기 불편하고. 그래서 그냥 만들었습니다.
